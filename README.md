# 🎮 Game Community Hub

A modern and interactive Game Community Hub website designed to bring gamers together in one place. The platform features dedicated pages for discovering games, participating in tournaments, tracking leaderboards, connecting with the community, and managing user profiles.

## ✨ Features
- 🏠 Home Page – Overview of the platform and featured content
- 🎮 Games Page – Explore and discover games
- 🏆 Tournament Page – View and participate in gaming tournaments
- 📊 Leaderboard Page – Track player rankings and scores
- 💬 Community Page – Connect and engage with fellow gamers
- 👤 Profile Page – Manage user information and gaming activities

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 Project Goal
To create a responsive and engaging gaming platform that serves as a centralized hub for gamers to explore, compete, and connect with the community.
